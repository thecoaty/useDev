# UseDev
Simulação da estrutura de um E-commerce desenvolvido nas aulas da trilha de CSS Avançado da Alura, no curso de CSS Grid. O projeto tem o objetivo de aprender e aplicar os conceitos de Display Grid, desenvolvendo uma pseudoplataforma com um design moderno.
